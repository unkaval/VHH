# `points_in_a_circle`
Uniform random points in a circle by radius, count and offset

## Inputs

### `point_count`
The number of points to scatter inside the circle.

### `radius`
Circle radius.

### `center`
Circle center.

### `seed`
Seed for random number generator.

### `axis`
Axis to construct circle upon.

## `Display`
### `display_size`
Display point size.

### `display_shape`
Display point shape.

## Output

### `point_positions`
Array of point positions.

### `out_points`
Pointcloud output.
